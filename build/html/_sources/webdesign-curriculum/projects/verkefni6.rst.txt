Verkefni 6
==========

Stærð skiptir máli

.. code-block:: javascript
    
    var tala1 = 12;
    var tala2 = 6;

    if(tala1 > tala2){
        console.log("Tala eitt er stærri en tala tvö");
    }
    else if(tala1 < tala2){
       console.log("Tala tvö er stærri en tala eitt");
    }
    else{
       console.log("Undir hvaða kringumstæðum keyri ég?");
    }
